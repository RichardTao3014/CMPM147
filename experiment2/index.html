Experiment 02: Living Impressions

Description

This generative sketch recreates a desert scene inspired by a photo I took while camping in Death Valley. It features mountains, sand, clouds, and bushes, with randomized layout each time the scene is reimagined.

Technical

The sketch uses p5.js with `beginShape()` for mountains and `ellipse()` for clouds and bushes. Randomness controls the position and scale of elements. The canvas updates each time the "Reimagine" button is clicked, using a seeded generation system to vary the composition.

Reflection

This project helped me explore how code can express natural landscapes. Translating real-world elements like clouds or dunes into generative visuals was both creative and technical. It was also meaningful to base the project on a photo I took myself.
