Experiment 3 - Alternate Worlds


Description

This project was all about trying out how to randomly generate different types of maps, like terrain and dungeons.

Technical

I used p5.js to create a grid-based world where each map is generated using a seed, then shown with tiles and matching ASCII characters.

Reflection

It was fun to figure out how code can turn simple rules into full maps—felt like building a little game world from scratch.
